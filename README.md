# VolansAviyonikKodlar

1) kullandığınız kütüphaneleri kodun en üst kısmına #include "kütüphane" şeklinde ekleyin
> [!WARNING]
> kullandıysanız değişkenleri setup kısmının üstünde tanımlayın. absürd alakasız isimler kullandıysanız bunları değiştirin.
3) setup kısmını ekleyin
4) loop kısmına hangi sensörü kullanıyorsanız onun anlık veri çekme kodunu ekleyin ( jiroskop ise ivme ve eğim ; GPS ise enlem , boylam , yükseklik hız vs gibi) ardından serial print , serial write kodu ile bu verileri "konsola" yazdırın.
*yazdığınız koda göre ben de eklediğiniz verileri paket haline getiren fonksiyonu ekleyeceğim.
